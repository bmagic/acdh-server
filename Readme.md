#ACDH.fr API Server
This code is the backend part of ACDH.fr. It's a simple REST API Server build on Nodejs.
## Requirements
- nodejs 
- mongodb

## Install 
```
npm install 
```

### Run 
```
npm start 
```

### Env Variables
- PORT (default: 4000)
- MONGO_URL (default: mongodb://localhost/acdh)
- LOG_LEVEL (default: info)
- SESSION_SECRET
- MAILJET_KEY  
- MAILJET_SECRET
